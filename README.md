# AI Cover Letter Generator

## Overview
The AI Cover Letter Generator is a React-based web application that creates personalized, ATS-friendly cover letters using Google's Gemini AI. Users can enter their personal details, upload their resume, and generate a professional cover letter tailored to a specific job role and company.

## Features
- Generate AI-powered cover letters
- Upload and parse PDF resumes
- Personalized cover letters using resume content
- Copy generated cover letter to clipboard
- Responsive design for mobile, tablet, and desktop
- Secure API key management using environment variables

## Tech Stack
- React.js
- Vite
- Tailwind CSS
- Google Gemini API
- PDF.js (`pdfjs-dist`)
- React Dropzone
